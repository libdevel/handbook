Before You Create that Guide...

General Subject Guides curate Library and other research-related resources oriented around prominent disciplines at Notre Dame.

Contact the LMT first...
General Subject guides are managed by the LibGuides Management Team. Please contact the LMT before creating a General Subject Guide.

Create the Guide

1 Click  Create Guide from your LibGuides homepage.
2 Choose Copy content/layout from an existing guide from the Choose Layout or Reuse dropdown.
3 Choose General Subject Template from the Select a Guide dropdown.
4 Type the guide name in the Guide Name field (e.g., 'Cultural Studies').
5 Choose Subject from the Guide Type dropdown.
6 Click   Create Guide

Add or Delete Pages

Before adding or deleting pages please note the requirements for information architecture for General Subject Guides.

1 Click the plus icon () under the left navigation to add a new page.
2 Select Delete Page from the Page dropdown to delete any page you're currently on

3

Add or Delete Boxes

1 Click the Add Box - Column 1 link (below the last box on the page) to add a box.
2 Enter the box title into the Box Name field (e.g., "The Collections")
3 Choose the [box type] from the Type dropdown. (For Mapped Boxes, see these instructions.)
4 Select the Draft Mode checkbox to temporarily hide a box from public view. Optional
Do not select the Floating Box checkbox. Floating boxes are not to be used in Library Guides.
5 Click  Save
Step 1
Steps 2-4

Assign Friendly URLs to All Pages

1 Click the pencil icon () next to the Guide URL to add a friendly URL to the guide.
2 Click the pencil icon () next to the Page URL to add a friendly URL to the current page.
Repeat step 2 for each page of the guide which doesn't already have a friendly URL.

Assign Subject(s) and Tags

1 Click the pencil icon () next to Subjects to associate the guide with 1 approved general subject.
2 Click the pencil icon () next to Tags to add up to 8 tags.

If you wish to associate the guide with more than 1 subject and/or 8 tags, please contact the LMT.

Publish/Unpublish the Guide

1 Click on the dropdown reflecting the current status of the guide. In our example it reads:  private
2 Select Change Status and Share
3 In the pop up window, select the Guide Publishing tab.
4 Select Unpublished, Published, or Private from the Publication Status dropdown.
