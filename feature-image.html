Image Builder

For simple image/text combinations, the Image Builder is far friendly alternative to using the system dialog for embedding images. The first step in working with images is to open the image repository. When uploading images, ensure you've saved your images at a sufficiently high resolution before uploading.

Pro Tip
Need to add a link in that Caption field? No worries—all textual input fields are HTML-friendly! You might type something like: Visit <a href="http://www.lautrec-online.com">Tolousse Lautrec Online</a> for more information!
