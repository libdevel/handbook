Standard lists

Standard lists can be generated from within the WYSIWYG editor.

Favor System Links When Creating Lists of Links
Springshare's linkchecker detects broken links for links added as System Content but cannot find broken links in Rich Text / HTML boxes. Therefore, lists should not be used to enumerate groups of library resources such as databases, books, and other general links. Enter these as System Content types.
