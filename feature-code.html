Code Wells

Code Wells are rendered with the <pre> element. Google's Prettify package is used to color-code the output.

Spacing Counts
Unlike other HTML elements, within the <pre> element, all spacing is preserved exactly as you type it.

HTML Converter

When pasting HTML or any code which uses angle brackets into the <pre> element, you must replace "<" and ">" with their corresponding HTML entities to render the code properly.

This converter will replace "<" and ">" in your HTML and surround it with <pre> tags.
