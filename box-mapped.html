Mapped Box

Certain boxes are commonly shared among Guide authors. By mapping to these relatively static boxes rather than copying them, we can reduce workload and ensure changes to content are propagated to all guides when the source box is changed. Mapped boxes are stored in the Reusable Assets Guide and include:

  • Books
  • Theses and Dissertations
  • Specific Search Widgets
  • Citation Managers
  • Citation Styles

Instructions for Mapping/Copying a Box

1

Browse Reusable Assets Guide

Browse the "Reusable Assets" guide to determine the box(es) you need to include in your guide.

See the handy link at the top right of the Handbook.

Navigate to Reusable Assets Guide

2

Select Reusable Assets Guide
  1. Click on Add Box – column 1
  2. Select the Reuse Existing Box tab in the pop-up window
  3. Select Reusable Assets from the Guide dropdown

Selecting the Reusable Assets Guide

3

Select the Box to Map to

Scroll through the dropdown until you find the page and box you wish to map to. Select it.

Selecting a Box

4

Opt to Copy (optional)

If you wish to modify the box, opt to copy it rather than map to it. Scroll down the pop up window and tick the Copy option.

Copying (and not mapping to) a Box
