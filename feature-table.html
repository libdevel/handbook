Accented Table Rows


                    
By adding a color class to any table row  <tr> or table data   <td> element, you can change the background color, and make better meaning for you users.
Accent colors include: active (grey),  success (green),  info (blue),  warning (yellow),  danger (red).


                    


                     

                       

                          This will produce a dark grey row
                          With two table data cells
                       

                    


                    


                    


                       


<tr class="active">
   <td>This will produce a dark grey row</td>
   <td>With two table data cells</td>
</tr>
